# Poll SaaS Future Features (Later Roadmap)

This file is intentionally for post-MVP planning.

## Product Features to Add Later
- Scheduled poll publishing and auto-close rules.
- Advanced poll logic (ranked choice, weighted votes, branching questions).
- Team accounts and shared poll ownership.
- Poll templates and reusable question sets.
- Bulk participant import from CSV.
- Reminder emails for non-responders.
- QR code generation for poll links.
- Custom branding (logo, colors, custom domain).
- Public poll discovery page (optional).
- Embeddable polls for external websites.

## Insights and Analytics
- Response trends over time.
- Participant completion/drop-off rates.
- Geo/device breakdown for responses.
- Export to CSV, Google Sheets, and Notion.
- A/B test different poll question wording.

## Access, Security, and Compliance
- SSO (Google, Microsoft, SAML).
- 2FA for admin and poll owners.
- Audit logs for account and poll changes.
- Data retention policies.
- GDPR/CCPA tooling (data export/delete request flow).

## Monetization (After MVP)
- Free tier with response limit.
- Pro tier with unlimited polls and branding.
- Business tier with teams, SSO, and audit logs.
- Usage-based pricing for high-volume polls.

## Platform and Integrations
- Slack, Teams, and Discord sharing.
- Zapier/Make webhook automation.
- CRM integrations (HubSpot/Salesforce).
- API keys for external app access.

## Nice-to-Have UX Upgrades
- Drag-and-drop poll builder.
- Better accessibility tooling and audits.
- Dark mode and theme presets.
- Mobile-first UI improvements.
